## 🌌 Overview
**Mathiverse** is a high-performance mathematical suite featuring automated tools for:
* **Calculus:** Derivatives and integrals with step-by-step logic.
* **Matrices:** Linear system solver and determinant calculator.
* **Statistics:** Dataset analysis and normal distribution plotting.

## 🛠️ Built With
* HTML5 / CSS3 (Glassmorphism UI)
* JavaScript (Mathematical Engines)
* Lucide Icons
